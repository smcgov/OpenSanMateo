INTRODUCTION
------------
  Integration with the ShareThis [http://sharethis.com/],
  social bookmarking utility on selected node types.

  * For a full description of the module, visit the project page:
    https://www.drupal.org/project/sharethis

  * To submit bug reports and feature suggestions, or to track changes:
    https://www.drupal.org/project/issues/sharethis


INSTALLATION
------------
  Install as you would normally install a contributed drupal module.
  @see https://www.drupal.org/documentation/install/modules-themes/modules-7
  for more details.


CONFIGURATION
-------------
  To configure module, navigate to
  "Admin => Configuration => Web services => ShareThis"
  Make required changes as per your site requirement and click on
  "Save Configuration" to update configuration.
